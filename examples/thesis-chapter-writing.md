# Example: Thesis Chapter Writing Loop

A real-world example for graduate students writing thesis chapters with AI assistance.

## Context

You're a PhD student. You need to write a thesis chapter. You have data, results, and a rough outline, but need help structuring and polishing the writing.

## The Loop (paste this into your AI tool)

### GOAL
Write a polished thesis chapter section (methodology or results) that is clear, well-structured, and meets academic writing standards.

### PLAN
1. Read the outline and understand which section to write.
2. Review the data/figures that will be referenced.
3. Write the first draft following the outline structure.
4. Self-review: check academic tone, logical flow, figure references.
5. Edit: fix awkward sentences, add transitions, verify citations.
6. Polish: grammar, formatting, consistency.

### EXECUTE
- Write section by section, not the whole chapter at once.
- After each section, verify: does it flow logically from the previous one?
- Reference figures and tables by number (Figure 1, Table 2).
- Use past tense for methods/results, present tense for established facts.

### VERIFY Checklist
- [ ] Each paragraph has a clear topic sentence
- [ ] Figures and tables are referenced in the text before they appear
- [ ] All acronyms are defined on first use
- [ ] No unsupported claims (every claim has a citation or data reference)
- [ ] Logical flow: problem → approach → results → discussion
- [ ] Consistent terminology throughout
- [ ] Word count is within advisor's requirements

### FIX Rules
- If a paragraph exceeds 8 sentences, split it.
- Replace "very" with specific quantifiers ("significantly", "marginally", "by 15%").
- If a sentence is > 40 words, break it in two.
- Never start consecutive paragraphs with the same word.

### STOP Conditions
```
STOP when:
- All sections in the outline are written
- All checklist items pass
- Two consecutive reviews find no issues

STOP and ASK when:
- Advisor's requirements are ambiguous
- Statistical interpretation requires domain expertise
- You're unsure if a claim is supported by the data
- Section structure doesn't match the department's template
```
