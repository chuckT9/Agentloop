# Writing Loop

Paste this into your AI tool. Give it a topic or draft. It will write, review, and polish automatically.

## Loop Specification

### GOAL
Produce a polished piece of writing that meets the specified requirements.

### PLAN
1. Understand the brief: topic, audience, tone, length, format.
2. Create an outline. Get approval before writing.
3. Write the first draft following the outline.
4. Self-review: check structure, flow, clarity.
5. Edit: fix issues found in review.
6. Polish: grammar, style, word choice.
7. Final check against the original brief.

### EXECUTE
- Draft in full before editing. Don't edit mid-draft.
- After each revision, show what changed (diff-style or summary).
- Track word count at each stage.

### VERIFY Checklist
- [ ] Meets the word count requirement (±10%)
- [ ] Matches the requested tone and audience
- [ ] Has a clear introduction, body, conclusion
- [ ] No repetition or filler
- [ ] All claims are accurate (flag unverifiable claims)
- [ ] Transitions between sections are smooth
- [ ] No grammar or spelling errors
- [ ] Formatting matches requirements (headers, lists, etc.)

### FIX Rules
- Read the draft OUT LOUD (mentally). If it sounds awkward, rewrite.
- Cut any sentence that doesn't earn its place.
- Replace jargon unless the audience expects it.
- If a paragraph exceeds 5 sentences, consider splitting.

### STOP Conditions
```
STOP when:
- All checklist items pass
- Word count is within range
- Two consecutive reviews find no issues

STOP and ASK when:
- Tone or audience requirements are ambiguous
- Factual claims cannot be verified
- Creative direction is unclear after draft
- Major restructuring is needed (confirm before rewriting)
```

### Output Format
```
## Writing Report

### Brief
- Topic: X
- Audience: X
- Tone: X
- Target length: X words

### Iteration Log
- Draft 1: X words, [issues found]
- Draft 2: X words, [fixes applied]
- Final: X words

### Final Version
[The polished piece]

### Self-Review Notes
- Strengths: X
- Areas considered but left as-is: X
```
